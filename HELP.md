# 超市管理系统（SupermarKetSystem）

### 技术栈

这是一个后端基于SpringBoot的框架制作，其中SpringBoot整合了springMVC、SpringSecurity、MybatisPlus等
前端上，基于Bootstrap框架和thymeleaf模版引擎的使用和JQuery库，以及前端三剑客html、css、javaScript
调试上，使用了git管理调试
